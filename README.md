# homebrew-tbreakout

Homebrew formula for installing [`tbreakout`](https://github.com/Geralt-Of-Rivia-Witcher/tbreakout), a retro terminal Breakout game written in Go using `tcell`.

## Main Game Repository

- https://github.com/Geralt-Of-Rivia-Witcher/tbreakout/

## Requirements

- macOS
- Homebrew

This formula installs prebuilt Darwin binaries for:

- Apple Silicon (`arm64`)
- Intel (`amd64`)

## Installation

```bash
brew tap Geralt-Of-Rivia-Witcher/tbreakout
brew install tbreakout
```

## Run

```bash
tbreakout
```

## Verify Version

```bash
tbreakout --version
```

## Update

```bash
brew update
brew upgrade tbreakout
```

## Uninstall

```bash
brew uninstall tbreakout
```
